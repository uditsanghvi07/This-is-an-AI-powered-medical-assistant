# This-is-an-AI-powered-medical-assistant
An AI-powered medical assistant that provides clear, reliable health information using LangChain, FAISS, and a HuggingFace LLM. It retrieves accurate context before answering and offers fast, user-friendly access to medical knowledge. For informational use only, not a substitute for professional advice.
